# 🔢 Exercícios e Algoritmos em Linguagem C

Este repositório contém uma coleção de programas desenvolvidos em Linguagem C como parte dos meus estudos acadêmicos em Engenharia da Computação no CEFET-MG e para prática pessoal de lógica de programação.

Cada arquivo foca em um conceito específico, partindo de operações básicas até implementações de algoritmos matemáticos e otimizações de fluxo.

## 🛠️ Conteudos e Conceitos Praticados

Aqui estão os principais tópicos de programação cobertos pelos exercícios deste repositório:

### 🟢 Nível Básico (Sintaxe, I/O e Operações)
* **Entrada e Saída de Dados:** Uso de `printf`, `scanf` e configuração de localidade com `setlocale`.
* **Operações Aritméticas:** Cálculos fundamentais com inteiros e reais.
* **Códigos:**
    * `calculadora_soma.c`: Solicita dois números inteiros e exibe a soma.
    * `quadrado_numero_real.c`: Calcula o quadrado de um número real (`float`).
    * `calculo_dobro_numero.c`: Solicita um número e retorna o seu dobro.

### 🟡 Nível Intermediário (Estruturas de Decisão e Strings)
* **Estruturas Condicionais:** Uso de `if`, `else` e `switch case`.
* **Manipulação de Strings:** Verificação de caracteres em arrays.
* **Códigos:**
    * `conceito_aluno_switch.c`: Classifica o desempenho acadêmico utilizando a estrutura `switch`.
    * `login_simples.c`: Simula uma autenticação verificando nome e senha caractere por caractere.

### 🔴 Nível Avançado (Algoritmos e Otimização)
* **Estruturas de Repetição:** Uso de loops `for` para cálculos iterativos.
* **Otimização Lógica:** Uso de vetores de strings para substituir múltiplos blocos condicionais, reduzindo a complexidade do código.
* **Algoritmos Matemáticos:** Implementação de métodos numéricos.
* **Códigos:**
    * `raiz_quadrada_newton.c`: Cálculo da raiz quadrada através do método de aproximação iterativa de Newton-Raphson.
    * `verificador_par_impar_vetor.c`: Identifica se um número é par ou ímpar utilizando o resto da divisão como índice de vetor.
    * `numero_negativo_zero_positivo_vetor.c`: Classifica números em três estados usando lógica matemática aplicada a índices de arrays.
